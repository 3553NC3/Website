    How I started Coding

    I want to be a computer software engineer when I grow up. I first decided I wanted to be a computer engineer when I was 14 years old. I have always been interested in computers and when I would look at a computer I would ask myself what can I make this do. I soon realized that this wasn’t normal and that most people asked themselves what does this do. I did some research to learn how computers work and how to make websites and apps. At the time I decided what I wanted to do when I grew up I wasn’t interested in creating things for others to use, I was interested in hacking and how to give myself an advantage over others. I started with hacking games by creating my own cheats instead of just downloading cheats others made. It took me about 2 months to understand the basics of exploiting weaknesses and then to be able to exploit those weaknesses. At 3 months into my research and progress I was able to create my own undetected cheats for games. I got bored with games and I didn’t know what else to do with my skills. I was wandering through Taylorsville and saw a poster saying advertising the Code to Success program. I decided if I am going to be a software engineer then I should know the coding languages such as html, CSS, javascript, C++, python, etc. I applied for the program and was accepted and now I am waiting for my classes to start. I completed the requirements early on before it started and I have gotten restless so I decided to start creating with what I knew and after 2 weeks I am almost finished writing the html, CSS, and javascript for my website. Coding is the future, everyone relies on code. Without code we don’t have computers, cell phones, etc. The world relies on code and knows that it exists but doesn’t know how it works. Firewalls, antivirus software, Google, Facebook, Snapchat, and any other program that has to do with a computer is built with code and I plan on helping the future by coding. There is a large demand for new coders because new coders know about all the new programs and what it takes to create them. I believe that I am one of the few that have what it takes to be a successful coder because I have started at a young age and I really enjoy coding. I am a hard working with a passion and that will make me successful.
